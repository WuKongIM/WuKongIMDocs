## 快速入门

### 安装

[![pub package](https://img.shields.io/pub/v/wukongimfluttersdk.svg)](https://pub.dartlang.org/packages/wukongimfluttersdk)

```yaml
dependencies:
  wukongimfluttersdk: ^version # 版本号看上面
```

### 引入

```dart
import 'package:wukongimfluttersdk/wkim.dart';
```
