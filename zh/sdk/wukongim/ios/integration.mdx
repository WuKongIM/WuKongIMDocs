## 通过 CocoaPods 集成

在项目的 `Podfile` 中添加依赖：

```objc
pod 'WuKongIMSDK'
```

然后执行安装命令：

```bash
pod install
```

## 通过源码集成

如果需要使用最新的开发版本，可以直接从 GitHub 仓库集成：

```objc
pod 'WuKongIMSDK', :git => 'https://github.com/WuKongIM/WuKongIMiOSSDK.git'
```
