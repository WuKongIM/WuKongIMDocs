## SDK 类型选择

  <Card title="WuKongEasySDK" icon="rocket" href="/zh/sdk/easy/overview">

## 快速对比

| 特性 | WuKongEasySDK | WuKongIMSDK |
|------|---------|-------------|
| **集成时间** | 5分钟 | 30分钟+ |
| **学习成本** | 很低 | 中等 |
| **功能完整性** | 核心功能 | 完整功能 |
| **自定义能力** | 有限 | 高度可定制 |
| **适用场景** | 快速原型、简单应用 | 企业级、复杂应用 |
